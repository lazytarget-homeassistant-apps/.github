## 🚀 Getting Started

Most repositories can be added directly via **HACS** (Home Assistant Community Store).

1. Open HACS in your Home Assistant instance
2. Go to **Integrations** (or **Frontend** for UI components)
3. Click the three-dot menu → **Custom repositories**
4. Paste the repository URL and select the appropriate category
    a. Stable versions: <https://github.com/lazytarget-homeassistant-apps/repository>
    b. Beta versions: <https://github.com/lazytarget-homeassistant-apps/repository-beta>
    c. Edge versions: <https://github.com/lazytarget-homeassistant-apps/repository-edge>
    d. ...or reference the App repo directly
5. Install and configure in Home Assistant

> Some repositories may support direct installation via the **Home Assistant Add-on Store** — check individual repo READMEs for details.
